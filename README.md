# 🚚 Swift Logistics — Transportation & Logistics Management System

## 📌 Project Overview

Swift Logistics is a full-stack web-based Transportation & Logistics Management System designed to simplify shipment booking, tracking, and payment processing.

This project was developed as a **B.Tech Minor Project** at ITM University.

---

## 🎯 Objectives

* Develop a logistics management system using Node.js & Express.js
* Implement secure user authentication using bcryptjs
* Provide multi-modal shipment booking (6 service types)
* Enable shipment tracking using AWB numbers
* Implement payment module with GST & insurance calculation

---
